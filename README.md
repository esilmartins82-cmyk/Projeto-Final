Projeto Co Piloto de Vendas FYS

Foi desenvolvido uma estratégia para
inserção da marca FYS em padarias
baseado no "COMBO MENOS CULPA", um 
banner escrito algo do tipo:"SE VOCE 
VAI COMER 800 CALORIAS DE UMA VEZ, PELO 
MENOS O REFRI É PREMIUM E TEM 50% DE 
AÇÚCAR. DE NADA."

A introdução traz o contexto, com as 
possibilidades desse Copiloto, e o 
projeto traz a estrutura, que pode ser 
usada como guia ou adaptado como prompt.
