# Projeto-Final
Projeto FYS de inserção em padaria através de combo

Iniciei uma chat no Gemini explicando 
como e o bootcamp do DIO, expliquei a dinâmica do projeto final e escrevi um prompt para analisar a marca FYS pelo site, focando na análise do tipo de comunicação usado pela marca e identificação do público alvo.

Em seguida criei o seguinte prompt:
Eu quero que você se comporte como um consultor de marketing e vendas. Nós vamos traçar um projeto no nível estratégico, tático e operacional para a inserção da marca Fys nas padarias do Brasil.

Fui reformulando a resposta fornecida colocando mais contexto, depois criei um arquivo PDF deste chat, junto com os arquivos dos repositórios do bootcamp do DIO e coloquei no Notebook LM para formulação completa do projeto.
